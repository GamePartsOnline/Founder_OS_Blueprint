# Politique de Permissions

| Action Sensible | Statut | Condition / Explication |
| :--- | :--- | :--- |
| **Envoyer un email réel au client** | ⚠️ Autorisé avec validation | L'agent prépare le brouillon (Gmail), l'humain clique sur Envoyer. |
| **Créer/Modifier l'arborescence d'un dossier** | ✅ Autorisé | Automatisation essentielle pour gagner du temps. |
| **Générer un devis ou une facture** | ✅ Autorisé | L'agent peut générer le PDF, mais pas l'envoyer directement. |
| **Supprimer un dossier client (Vault)** | ❌ Interdit | Sécurité des données : seul le Founder peut supprimer. |
| **Utiliser une API payante** | ⚠️ Autorisé avec validation | Nécessite une limite de budget (cap) définie par l'humain. |
| **Modifier le statut Kanban d'un dossier** | ✅ Autorisé | L'agent gère le suivi visuel de l'avancement. |

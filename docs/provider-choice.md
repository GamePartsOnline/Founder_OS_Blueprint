# Choix du Provider (Local vs Cloud)

**Décision provisoire : Approche 100% Cloud (Mixte).**

**Justification :** 
L'objectif pour GPO SARL est de s'appuyer sur des solutions simples, gratuites (ou très accessibles) et facilement interconnectables avec des outils existants comme Gmail.
- Je vais utiliser un modèle **Cloud** (comme OpenAI, Anthropic Claude, ou Google Gemini) via des requêtes API ou des plateformes no-code.
- Cela me permet d'éviter les contraintes matérielles d'une machine locale, tout en profitant d'une intelligence capable de comprendre la complexité des 5 services par dossier de GPO.
- La mémoire et le stockage (Vault) resteront sur le Cloud (ex: Google Workspace) pour faciliter le suivi informatique partout.
